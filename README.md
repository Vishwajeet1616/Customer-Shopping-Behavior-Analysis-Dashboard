# 📊 Customer Shopping Behavior Analysis & Dashboard

This project represents a complete, industry-standard, end-to-end data analytics workflow. It encompasses all critical stages of data analysis, from data preparation and exploratory data analysis (EDA) to SQL-based insight generation and interactive Power BI reporting.

**Perfect for:**
- 📈 Building a strong **Portfolio Project** demonstrating real-world analytics skills for interviews.
- 📚 Showcasing proficiency in **Python, SQL, and Power BI**.
- 💼 Demonstrating the ability to translate raw data into strategic business intelligence.

---

## 📌 Project Overview
The goal of this project is to simulate a corporate-grade data analytics workflow, translating raw customer data into actionable business intelligence:

✅ **Data Preparation & EDA (Python):** Cleaned, transformed, and explored the raw dataset to understand initial trends and distributions.
✅ **Data Analysis (SQL):** Extracted deep insights on customer segments, purchasing patterns, and revenue metrics by querying the structured data.
✅ **Visualization & Insights (Power BI):** Built an interactive dashboard highlighting key patterns, demographic impacts, and sales performance for stakeholders.

## 📂 Repository Structure
* **`customer_shopping_behavior.csv`**: The raw customer transaction dataset.
* **`customer_shopping_behavior.ipynb`**: Jupyter Notebook containing the Python code for data cleaning and EDA.
* **`customer_shopping.sql`**: SQL scripts utilized for data extraction and business querying.
* **`customer_shopping_dashboard.pbix`**: Interactive Power BI dashboard file.
